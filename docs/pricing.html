Frequently asked questions

How does the license key work?

After payment you'll receive a license key starting with EK-PRO-, EK-YEAR-, or EK-LTD-. Open the extension popup → Menu → Settings → License Key, paste it in, and click Activate. That's it — all Pro features unlock immediately.

What is the 50-recording free tier limit?

The free tier lets you record up to 50 unique API interactions per IndexedDB session. Use "Clear recordings" from the menu to start fresh and record another 50. This is designed to be generous enough for quick debugging while encouraging Pro for heavy-use workflows.

Can I use one license on multiple machines?

Yes. Your license key can be activated on up to 3 devices simultaneously (browser profiles). Chrome Sync will keep the key in sync across your signed-in Chrome profile automatically.

Is there a refund policy?

Yes — 14-day no-questions-asked refund for all plans. Contact support via GitHub Issues.

Does the Lifetime plan cover future updates?

Yes. Every feature added to EchoKit is included in your Lifetime license at no extra cost.
